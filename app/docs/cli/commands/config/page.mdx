# `kubetail config`

```console
Subcommands for initializing and modifying the kubetail configuration.

Usage:
  kubetail config [command]

Available Commands:
  init        Initialize a default configuration file

Flags:
  -h, --help   help for config

Global Flags:
      --kubeconfig string   Path to kubeconfig file
      --in-cluster          Use in-cluster Kubernetes configuration
  -c, --config string       Path to config file (default is $HOME/.kubetail/config.yaml)

Use "kubetail config [command] --help" for more information about a command.
```
