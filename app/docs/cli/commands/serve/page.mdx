# `kubetail serve`

```console
This command starts the dashboard server and opens the UI in your default browser.

Usage:
  kubetail serve [flags]

Flags:
  -p, --port int           Port number to listen on (default 7500)
      --host string        Host address to bind to (default "localhost")
  -l, --log-level string   Log level (debug, info, warn, error, disabled) (default "info")
      --skip-open          Skip opening the browser
      --test               Run internal tests and exit
  -h, --help               help for serve
```
