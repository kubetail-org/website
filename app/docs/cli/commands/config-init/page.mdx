# `kubetail config init`

```console
This command creates a default configuration file.

Usage:
  kubetail config init [flags]

Flags:
      --path string   Target path for configuration file (default is $HOME/.kubetail/config.yaml)
      --force         Overwrite existing configuration file
  -h, --help          help for init

Global Flags:
      --kubeconfig string   Path to kubeconfig file
      --in-cluster          Use in-cluster Kubernetes configuration
  -c, --config string       Path to config file (default is $HOME/.kubetail/config.yaml)
```
