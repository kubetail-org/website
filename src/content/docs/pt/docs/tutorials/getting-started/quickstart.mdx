## Instalação

A ferramenta CLI `kubetail` é o principal ponto de entrada do Kubetail no seu desktop. Para instalá-la, você pode usar o seu gerenciador de pacotes favorito:

<Tabs syncKey="os">
    <TabItem label="macOS">
        **Homebrew**
        ```sh
        brew install kubetail
        ```

        **Krew**
        ```sh
        kubectl krew install kubetail
        ```

        **MacPorts**
        ```sh
        sudo port install kubetail
        ```

        **Nix (Flake)**
        ```sh
        nix profile add github:kubetail-org/kubetail-nix
        ```

        **Nix (Classic)**
        ```sh
        nix-env -i -f https://github.com/kubetail-org/kubetail-nix/archive/refs/heads/main.tar.gz
        ```

        **asdf**
        ```sh
        asdf plugin add kubetail https://github.com/kubetail-org/asdf-kubetail.git
        asdf install kubetail latest
        ```
    </TabItem>
    <TabItem label="Linux">
        **Homebrew**

## Uso

Depois de instalar, você já pode começar a fazer tail dos logs. A ferramenta CLI `kubetail` usa o seu arquivo kubeconfig para autenticação, então você poderá acessar os mesmos clusters que o `kubectl`. Consulte a [referência da CLI](/pt/docs/reference/cli) para ver a lista completa de comandos.

* **Dashboard web**

    Inicie o dashboard web usando o comando [`serve`](/pt/docs/reference/cli#kubetail-serve) (ele abrirá em [http://localhost:7500](http://localhost:7500)):

    ```sh
    kubetail serve
    ```

* **Terminal**

    Faça streaming de logs diretamente no terminal com o comando [`logs`](/pt/docs/reference/cli#kubetail-logs):

    ```sh
    kubetail logs -f deployments/my-app
    ```

* **Configuração**

    Crie um arquivo de configuração local em `~/.kubetail/config.yaml` usando o comando [`config init`](/pt/docs/reference/cli#kubetail-config):

    ```sh
    kubetail config init
    ```

* **Recursos avançados**

    Instale recursos do lado do cluster usando o comando [`cluster install`](/pt/docs/reference/cli#kubetail-cluster) para habilitar recursos avançados, como busca:

    ```sh
    kubetail cluster install
    ```
