Para executar a ferramenta CLI do Kubetail (`kubetail`) dentro de um contêiner, voce pode usar a imagem Docker [`kubetail-cli`](https://hub.docker.com/r/kubetail/kubetail-cli). A imagem e publicada no Docker Hub e no GitHub Container Registry a cada release:

| Registry   | Image                               |
|------------|-------------------------------------|
| Docker Hub | `kubetail/kubetail-cli`             |
| GHCR       | `ghcr.io/kubetail-org/kubetail-cli` |

Ha duas variantes disponiveis:

| Sufixo da tag | Imagem base | Descricao |
|---------------|-------------|-----------|
| _(nenhum)_ | `scratch` | Imagem minima, menor footprint |
| `-alpine` | `alpine` | Inclui shell e utilitarios padrao |

Executar `kubetail` dentro do Docker permite integrar o Kubetail aos seus fluxos locais de desenvolvimento com Docker e tambem depurar seus logs de dentro do cluster.
