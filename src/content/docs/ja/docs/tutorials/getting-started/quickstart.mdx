## インストール

`kubetail` CLI ツールは、デスクトップで Kubetail を使うための主要なエントリーポイントです。インストールには、お好みのパッケージマネージャーを使用できます。

<Tabs syncKey="os">
    <TabItem label="macOS">
        **Homebrew**
        ```sh
        brew install kubetail
        ```

        **Krew**
        ```sh
        kubectl krew install kubetail
        ```

        **MacPorts**
        ```sh
        sudo port install kubetail
        ```

        **Nix (Flake)**
        ```sh
        nix profile add github:kubetail-org/kubetail-nix
        ```

        **Nix (Classic)**
        ```sh
        nix-env -i -f https://github.com/kubetail-org/kubetail-nix/archive/refs/heads/main.tar.gz
        ```

        **asdf**
        ```sh
        asdf plugin add kubetail https://github.com/kubetail-org/asdf-kubetail.git
        asdf install kubetail latest
        ```
    </TabItem>
    <TabItem label="Linux">
        **Homebrew**
        ```sh

        ```

        **Krew**
        ```sh
        kubectl krew install kubetail
        ```

        **Snap**
        ```sh
        sudo snap install kubetail
        ```

        **Ubuntu/Mint (apt)**
        ```sh
        sudo add-apt-repository ppa:kubetail/kubetail
        sudo apt update && sudo apt install kubetail-cli
        ```

## 使い方

インストールが終われば、すぐにログの tail を始められます。`kubetail` CLI ツールは認証に kubeconfig ファイルを使うため、`kubectl` と同じクラスターにアクセスできます。すべてのコマンド一覧は [CLI リファレンス](/ja/docs/reference/cli) を参照してください。

* **Web ダッシュボード**

    [`serve`](/ja/docs/reference/cli#kubetail-serve) コマンドを使って Web ダッシュボードを起動します（[http://localhost:7500](http://localhost:7500) で開きます）:

    ```sh
    kubetail serve
    ```

* **ターミナル**

    [`logs`](/ja/docs/reference/cli#kubetail-logs) コマンドを使ってログをターミナルへ直接ストリームします:

    ```sh
    kubetail logs -f deployments/my-app
    ```

* **設定**

    [`config init`](/ja/docs/reference/cli#kubetail-config) コマンドを使って、`~/.kubetail/config.yaml` にローカル設定ファイルを作成します:

    ```sh
    kubetail config init
    ```

* **高度な機能**

    [`cluster install`](/ja/docs/reference/cli#kubetail-cluster) コマンドでクラスター側のリソースをインストールし、検索などの高度な機能を有効にします:

    ```sh
    kubetail cluster install
    ```
