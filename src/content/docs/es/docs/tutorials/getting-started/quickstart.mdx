## Instalación

La herramienta CLI `kubetail` es el principal punto de entrada a Kubetail en su escritorio. Para instalarla, puede usar su gestor de paquetes favorito:

<Tabs syncKey="os">
    <TabItem label="macOS">
        **Homebrew**
        ```sh
        brew install kubetail
        ```

        **Krew**
        ```sh
        kubectl krew install kubetail
        ```

        **MacPorts**
        ```sh
        sudo port install kubetail
        ```

        **Nix (Flake)**
        ```sh
        nix profile add github:kubetail-org/kubetail-nix
        ```

        **Nix (Classic)**
        ```sh
        nix-env -i -f https://github.com/kubetail-org/kubetail-nix/archive/refs/heads/main.tar.gz
        ```

        **asdf**
        ```sh
        asdf plugin add kubetail https://github.com/kubetail-org/asdf-kubetail.git
        asdf install kubetail latest
        ```
    </TabItem>
    <TabItem label="Linux">

## Uso

Una vez instalada, ya puede empezar a hacer tail de logs. La herramienta CLI `kubetail` usa su archivo kubeconfig para autenticarse, por lo que podrá acceder a los mismos clústeres que con `kubectl`. Consulte la [referencia de la CLI](/es/docs/reference/cli) para ver la lista completa de comandos.

* **Dashboard web**

    Inicie el dashboard web usando el comando [`serve`](/es/docs/reference/cli#kubetail-serve) (se abrirá en [http://localhost:7500](http://localhost:7500)):

    ```sh
    kubetail serve
    ```

* **Terminal**

    Transmita logs directamente en su terminal con el comando [`logs`](/es/docs/reference/cli#kubetail-logs):

    ```sh
    kubetail logs -f deployments/my-app
    ```

* **Configurar**

    Cree un archivo de configuración local en `~/.kubetail/config.yaml` usando el comando [`config init`](/es/docs/reference/cli#kubetail-config):

    ```sh
    kubetail config init
    ```

* **Funciones avanzadas**

    Instale recursos del lado del clúster usando el comando [`cluster install`](/es/docs/reference/cli#kubetail-cluster) para habilitar funciones avanzadas, como la búsqueda:

    ```sh
    kubetail cluster install
    ```
