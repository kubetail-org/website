title: Docker Compose
description: "Utilisez Docker Compose avec l'image `kubetail-cli` pour monter votre kubeconfig et demarrer le serveur de dashboard Kubetail sur localhost:7500."
---

import { Aside } from '@astrojs/starlight/components';

Pour utiliser l'outil CLI localement avec `docker-compose`, vous pouvez partir de ce fichier de configuration [docker-compose.yml](https://raw.githubusercontent.com/kubetail-org/kubetail/refs/heads/main/config/examples/docker-compose.yml):

```sh
curl -LO https://raw.githubusercontent.com/kubetail-org/kubetail/refs/heads/main/config/examples/docker-compose.yml
docker-compose up
```

Cela montera votre fichier local `.kube/config` dans un conteneur `kubetail-cli` et demarrera le serveur de dashboard sur [http://localhost:7500](http://localhost:7500).

<Aside type="tip">
Vous pouvez aussi utiliser l'image Docker `kubetail-cli` avec:

* [Docker Run](/fr/docs/guides/docker/docker-run)
* [Kubernetes Pod](/fr/docs/guides/docker/kubernetes-pod)
</Aside>
