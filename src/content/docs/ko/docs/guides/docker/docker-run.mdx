CLI 도구를 `docker run`으로 로컬에서 사용하려면 로컬 `.kube/config` 파일을 `kubetail-cli` 컨테이너에 마운트해야 합니다.

```sh
docker run --rm -it \
  -v ~/.kube/config:/root/.kube/config:ro \
  kubetail/kubetail-cli logs \
  --kubeconfig /root/.kube/config \
  -f \
  deployments/my-app
```

`sh` 엔트리포인트를 사용해 대화형 세션을 시작할 수도 있습니다.

```sh
docker run --rm -it \
  -v ~/.kube/config:/root/.kube/config:ro \
  --entrypoint sh \
  kubetail/kubetail-cli
```

<Aside type="tip">
`kubetail-cli` Docker 이미지는 다음 방식으로도 사용할 수 있습니다.

* [Docker Compose](/ko/docs/guides/docker/docker-compose)
* [Kubernetes Pod](/ko/docs/guides/docker/kubernetes-pod)
</Aside>
