---
title: Minikube
description: "Kubetail을 Minikube의 내장 addon(1.36.0부터 제공)으로 한 번의 명령으로 설치한 뒤 `minikube service`로 대시보드를 열 수 있습니다."
---

[minikube](https://minikube.sigs.k8s.io/docs/) v1.36.0부터 Kubetail을 addon으로 설치할 수 있습니다.

```sh
minikube addons enable kubetail
```

클러스터에서 Kubetail Pod가 실행되면 service를 통해 접근할 수 있습니다.

```sh
minikube service -n kubetail-system kubetail-dashboard
```
